Print Square Module
====================
This module prints a square using the character #

Import function
---------------

>>> print_square = __import__('4-print_square').print_square

Normal square (size = 4)
------------------------

>>> print_square(4)
####
####
####
####

Square size 1
-------------

>>> print_square(1)
#

Empty square (size = 0)
-----------------------

>>> print_square(0)
