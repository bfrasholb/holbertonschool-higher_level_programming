Sorted List Module
==================
This module sorts then prints a list.

Import Function
---------------
>>> my_list = __import__('1-my_list').MyList

Instantiation with Empty List
-----------------------------
>>> test = my_list()
>>> test.print_sorted()
[]

Returns a Unique List
---------------------
>>> test_list = [1, 2, 3, 5, 4]
>>> test = my_list(test_list)
>>> test.print_sorted()
[1, 2, 3, 4, 5]
>>> print(test)
[1, 2, 3, 5, 4]

Sorting is correct with Negatives
---------------------------------
>>> test_list = [-1, -2, 3, 4, 5]
>>> test = my_list(test_list)
>>> test.print_sorted()
[-2, -1, 3, 4, 5]

Disorded Appends Become Sorted
------------------------------
>>> test.append(7)
>>> test.append(0)
>>> test.append(-23)
>>> test.print_sorted()
[-23, -2, -1, 0, 3, 4, 5, 7]
